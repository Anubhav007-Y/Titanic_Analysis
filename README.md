Titanic Survival Analysis Dashboard
Project Overview

This project analyzes the Titanic passenger dataset using Python and Power BI.

The workflow includes:

Data Cleaning using Pandas
Exploratory Data Analysis (EDA)
Data Visualization
Interactive Power BI Dashboard
Tools Used
Python
Pandas
NumPy
Matplotlib
Power BI
Dashboard KPIs
Total Passengers
Total Survivors
Survival Rate
Average Age
Average Fare
Dashboard Features
Gender slicer
Passenger Class slicer
Embarked slicer
Age Distribution
Survival by Gender
Survival by Passenger Class
